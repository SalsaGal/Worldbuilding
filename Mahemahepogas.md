# Mahemahespogas

**Mahemahespogas** (/maʃɛmaʃɛspoŋas/) is the language spoken by the **Atskenas**. The language name literally means "The language I'm speaking now" and is usually abbreviated to "mahegas"

## Phonology

### Consonants

|                 | Labial | Alveolar    | Velar   |
| --------------- | ------ | ----------- | ------- |
| **Plosive**     | p      | t           | k       |
| **Nasal**       | m      | n           | ŋ \<g\> |
| **Fricative**   |        | s z ʃ \<h\> |         |
| **Approximant** |        | l           |         |
